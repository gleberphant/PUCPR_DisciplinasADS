# objetivo do sistema 
gerar previsões orçamentárias para a empresa,

# Requisitos funcionais

* time financeiro -  inserir informações de todo o gasto da empresa com colaboradores, TI, produtos de higiene, produtos de escritório, custos fiscais da Receita Federal e orçamento aprovado do ano.
* time financeiro - alterar os valores ao longo do tempo para que a composição final de valores fique abaixo do orçamento aprovado.

* sistema -  guardar o histórico dessa simulação para que no ano seguinte, esse histórico entre no cálculo para alertar, por exemplo, que o valor da TI duplicou de um ano para outro
* sistema - gerar um alerta por e-mail para os gerentes responsáveis darem um parecer na própria ferramenta.
* sistema -  enviar dados finais ajustados devem  para o ERP da empresa, ou seja, o sistema principal para registro geral e oficial.

# Requisitos não funcionais

* desenvolvimento nos servidores do ERP da empresa
* capacidade de resposta rápida com pelo menos 20 usuários simultâneos
