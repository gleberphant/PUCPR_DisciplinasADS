# Visão do projeto

Esta atividade tem como objetivo consolidar os conhecimentos adquiridos na disciplina por meio da criação de um dashboard interativo no Power BI. Para isso, trabalharemos com base em um dataset de uma empresa chamada IAN-Store, realizaremos análises descritivas e diagnósticas, aplicaremos técnicas de visualização e storytelling, e implementaremos um modelo preditivo simples. Por fim, elaboraremos um relatório para destacar os principais insights e as aplicações práticas para o contexto de negócios.
