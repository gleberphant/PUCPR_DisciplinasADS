# DESCRIÇÃO DAS TAREFAS DE CADA SEMANA 

## Atividade formativa da semana 2: Implementar listas encadeadas 

Tarefa: Criar uma lista encadeada para gerenciar os livros na biblioteca. Cada nó da lista deve conter informações como título do livro, autor e ano de publicação. 

Aplicação prática: Usar a lista para armazenar e acessar a lista de livros. 

## Atividade formativa da semana 3: Implementar filas e pilhas 

Tarefa: Implementar uma fila para gerenciar uma lista de espera de livros (quando um livro está emprestado) e uma pilha para armazenar o histórico de navegação do usuário (último livro visualizado no topo). 

Aplicação prática: Usar a fila para gerenciar a espera de livros e a pilha para o histórico de navegação dos usuários. 


## Atividade somativa da semana 4: Implementar grafos 

Tarefa: Criar um grafo em que cada nó representa um livro e as arestas representam relações, como livros recomendados. 

Aplicação prática: Usar grafos para sugerir livros aos usuários com base em suas leituras anteriores ou interesses. 



## Atividade formativa da semana 5: Implementar árvores binárias 

Tarefa: Construir uma árvore binária de busca para categorizar livros por diferentes critérios, como gênero ou autor. 

Aplicação prática: Facilitar a busca rápida de livros por categoria específica. 



## Atividade formativa da semana 6: Implementar algoritmos de ordenação 

Tarefa: Implementar bubble sort e compará-lo com o merge sort para ordenar listas de livros por diferentes critérios. 

Aplicação prática: Avaliar a eficiência de diferentes algoritmos de ordenação em listas de livros. 



## Atividade formativa da semana 7: Implementar DFS ou BFS em árvores 

Tarefa: Utilizar DFS e BFS para explorar a árvore binária de livros, permitindo descobrir novas recomendações de leitura ou caminhos entre diferentes gêneros. 

Aplicação prática: Melhorar o sistema de recomendação de livros com base nas conexões entre eles. 



## Atividade somativa da semana 8: Implementar o algoritmo de Dijkstra em grafos 

Tarefa: Aplicar o algoritmo de Dijkstra para encontrar o caminho mais curto ou mais relevante entre diferentes livros ou categorias. 

Aplicação prática: Otimizar as recomendações para os usuários, levando em conta não apenas a similaridade, mas também a popularidade ou a avaliação dos livros.
