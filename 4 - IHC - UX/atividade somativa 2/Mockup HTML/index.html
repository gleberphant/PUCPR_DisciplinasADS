produtosDesenvolvimento

Página inicial do aplicativo que lista todos os produtos. mostra barra de progresso do backlog.

1. ao clicar no icone visualizar, abre a tela de visualização do produto

2. ao clicar no botão de arquivamento, arquiva o produto

3. ao clicar na arquivados mostra os itens que ja foram arquivados

4. botão para criar um novo produto . abre tela de novo produto

5. no input pesquisar o usuário poderá pesquisar qualquer campo apresentado na tela

6. botão exportar converte a  lista em um formato exce

dialogArquivamento

antes de arquivar abre tela para confirmação do arquivamento

produtosArquivados

tela para listar todos produtos que ja foram arquivados.

1. botão para desarquivar, envia o produto para lista de produtos em desenvolvimento

2. botão deletar abre caixa para confirmar deletar o produto

3. botão visualizar , abre a tela do produto

4. no input pesquisar o usuário poderá pesquisar qualquer campo apresentado na tela

5. botão exportar converte a  lista em um formato excel.

dialogDeletar

caixa de dialogo para usuario confirmar se realmente quer deletar

novoProduto

pagina para o usuário criar um novo produto

ele deverá preencher todos os campos antes de salvar o produto

Projeto

tela inicial de visualização de um produto.

1. ícone do lápis permite editar um determinado campo

Backlog

tela que apresenta o backlog do produto.

1. ícone do lápis permite o usuario editar a funcionalidade, perfil e estória.

2. combo prioridade o usuário escolhe entre alta, media e baixa

3. combo complexidade o usuário escolhe entre alta, media e baixa

4. combo status o usuário escolhe entre pendente, desenvolvimento e feito
