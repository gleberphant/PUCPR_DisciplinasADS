Camel Case -> JAVA
Camel Case é um estilo de nomenclatura comum, em que a primeira letra da palavra é minúscula e as palavras seguintes iniciam-se com maiúscula. 

Exemplo de aplicação: nomeDoUsuario, idadeDoUsuario.

Pascal Case -> C#
Semelhante ao Camel Case, a nomenclatura de Pascal Case difere porque todas as palavras são iniciadas com letras maiúsculas, incluindo a primeira letra. 

Exemplo de aplicação: NomeDoUsuario, IdadeDoUsuario.

Snake Case -> python
A nomenclatura Snake Case usa sublinhado para separar palavras, e todas as letras são geralmente minúsculas. 

Exemplo de aplicação: nome_do_usuario, idade_do_usuario.

Kebab Case -> CSS
É um estilo semelhante ao Snake Case, cuja a diferença está no separador usado, que utiliza hífens ao invés de sublinhados. 

Exemplo de aplicação: nome-do-usuario, idade-do-usuario.

Upper Case
Este estilo de nomenclatura utiliza todas as letras maiúsculas e, frequentemente, separadas por sublinhados. UPPER_CASE é bastante aplicado para nomear constantes em linguagens como C e C++.

Exemplo de aplicação: NOME_DO_USUARIO, IDADE_DO_USUARIO.
